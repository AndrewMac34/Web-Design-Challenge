Summary

The purpose of this assignment was to study weather patterns and see if they change and alter as one travels closer to the equator. An Open Weather API and the CitiPy Python Library were utilized to randomly pull 500 cities from around the world.

After assembling the data, Pandas and Matplotlib Python libraries were used to further examine the subject. The four dependent variables were: Max Temperature, Humidity, Cloudiness, and Wind Speed. This webpage shows the csv raw data and the charts based on each of the four dependent variables.
